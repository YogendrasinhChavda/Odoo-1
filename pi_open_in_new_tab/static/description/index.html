PI Open Record In New Tab

These set of features will enable you to open any
record in a new tab.


Features

  • 1. Open a single record in a new tab.
  • 2. Select and open multiple records in new tabs.
  • 3. Open one2many record in a new tab.


How to Use?

  • After installing this module you will see this icon in every tree view.
  • Click on that icon to open that record in a new tab.
  • If you want to select and open multiple records at once on a new tab, then you can do that by selecting them and clicking on the button "Open in New Tab". Refer the screenshots for details.


Note
  • Only 8 records can be opened at once using the Bulk-Open Feature. This restriction is to avoid enabling too many tabs. However you can modify the limitation in code.
  • When performing Bulk-Open for the first time, the browser will ask for permission to allow pop-ups. You need to allow permissions only for this url. Enabling all Pop-ups may be risky. Refer screenshots and video for more.

Screenshots

1. You can see this icon at the right corner in every tree view.


2. You can select multiple records (max 8) and click on "Open in new Tab" and all those selected records will open in new tabs.
